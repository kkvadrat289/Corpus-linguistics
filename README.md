## Разметка корпуса
#### Сегментация
python + nltk + russian language support (https://github.com/Mottl/ru_punkt)

Храниение: sentence.csv - каждому посту сопоставлен набор координат предложений

#### Токенизация и pos-tagging
python + nltk + pymorphy2

Хранение: json с координатами и грамматическими категориями

## Поиск

python + whoosh

* Время индексации: ~ 10минут
* Индексируемые поля:
  * Название
  * Активность
  * Автор
  * Наличие фото/изображения
  * Категория
  * Место
  * Рейтинг у пользователей

## Классификация текстов по активности

* TF-IDF + Logistic Regression

Результаты для самых объемных активностей:

|Активность|precision|recall|F1|support|
|---|---|---|---|---|
|Альпинизм|0.68|0.96|0.79|2253|
|Скалолазание|0.90|0.77|0.83|719|
|Горный туризм|0.59|0.32|0.42|288|
|Путешествия|0.55|0.35|0.43|271|
|Бэккантри/Фрирайд|0.72|0.56|0.63|219|

Вероятно, низкое качество обусловлено расплывчатой границей между категориями. Например, альпинизм, горный туризм, скалолазание имеют много общих черт и общих слов для описания (в среднем, люди, далёкие от этой области, их вообще не различают).
